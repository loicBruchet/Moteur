# - Moteur ⚙️

> **Statut :** Démo
> 
> **Type :** Moteur de jeu + Rail Shooter
>
> **Durée du projet :** 4 semaines

## 📖 Introduction
Ce projet est sans doute le plus compliqué que j'ai eu avec mon groupe à faire. L'objectif ici est donc de réaliser un moteur de jeu avec directX12 permettant, comme Unity ou Unreal, de créer un jeu.

## 🛠️ Stack Technique
* **Moteur :** OperatorEngine (nom du projet)
* **Langage :** C++
* **Outils de versionning :** GitHub

## 🧠 Ce que j'ai appris (Key Learnings)

### 1. concevoir un moteur

Concevoir tous les outils pour la création d'un jeu tel que l'ECS (entity component system), le rendu

### 2. Perserverer

Ce projet, étant très compliqué, la perseverence est essentiel pour aller jusqu'au bout de notre projet
